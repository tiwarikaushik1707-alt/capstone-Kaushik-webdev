LearnPearl – Online Learning Website

This is the Capstone Project for mu Web Development Lab Assignment.
I chose Online Learning Platform theme and created a quite simple responsive website called 'LearnPearl' using HTML and CSS3.


Project Description-

LearnPearl is a basic one-page website where users can see:

A header with navigation

A hero/banner section

A small “Why LearnPearl?” about section

A courses section with simple cards

A contact/enrollment form

A footer

The whole website is made using semantic HTML and simple CSS styling.

Features Included

Semantic structure (header, nav, main, section, footer, article)

Responsive layout using Flexbox and CSS Grid

Three main content sections

A working contact form layout

Skip-to-content link for accessibility

Simple hover styles

Media queries for mobile screens

Breakpoints Used

Mobile: up to 600px

Default/Desktop: above 600px

The design adjusts automatically on smaller screens.


Technologies Used-

HTML

CSS

Grid

Media  Queries

No JS.
